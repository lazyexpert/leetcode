# 119. Pascal's Triangle II
Given an integer `rowIndex`, return the `rowIndexth` (0-indexed) row of the Pascal's triangle.

## Constraints
- 0 <= rowIndex <= 33

## Example
```
Input: rowIndex = 3
Output: [1,3,3,1]
```

```
Input: rowIndex = 0
Output: [1]
```

```
Input: rowIndex = 1
Output: [1,1]
```
